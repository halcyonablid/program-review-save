# Mermaid Floating Widget

Obsidian plugin launcher for the Mermaid floating widget.

What it does:
- Adds a left ribbon icon to toggle the Mermaid floating widget
- Adds a command: `Toggle Mermaid floating widget`
- Loads the widget logic from a source note script

Default source note:
- `做mermaid图的悬浮窗.md`

Notes:
- The plugin does not duplicate the Mermaid widget logic
- It executes the source note's Templater-style script directly
- If you move or rename the note, update the plugin setting `Source note path`
